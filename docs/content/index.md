# srsRAN Project Documentation

![image](.imgs/srsran_arch_5g.png)

srsRAN Project is an open-source 5G CU/DU from [SRS](https://www.srs.io).

It is a complete RAN solution compliant with 3GPP and O-RAN Alliance specifications. srsRAN Project  includes the full L1/2/3 stack with minimal external dependencies. The software is portable across processor architectures and
scalable from low-power embedded systems to cloudRAN, providing a powerful platform for mobile wireless research and development.

Get started with srsRAN Project :

> - [Install](user_manuals/source/installation.md#manual-installation) srsRAN Project on your computer.
> - Get up and running with srsRAN Project [here](user_manuals/source/running.md#manual-running).
> - More advanced users should read the [Developers Guide](dev_guide/source/index.md#dev-guide).
> - Get up to speed on 5G and OpenRAN with our [Knowledge Base](knowledge_base/source/index.md#knowledge-base).

Useful resources:

> * srsRAN Project  [source code](https://github.com/srsran/srsRAN_project).
> * Explore the srsRAN Project  [Discussions](https://github.com/srsran/srsran_project/discussions) for news and user support.
> * Discover the [srsRAN Enterprise solution](https://srs.io/srsran-enterprise-5g/) for Private 5G network deployments.
